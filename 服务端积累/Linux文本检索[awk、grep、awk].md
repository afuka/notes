# 一、grep 基础语法



## 1. 常用参数

- `-n`  :显示行号
- `-o` :只显示匹配的内容
- `-C` :如果匹配成功，则将匹配行及其前后n行一起打印出来
- `--color` :带颜色输出
- `-i`  ：忽略大小写
- `-v`  :取反，不匹配



## 2. 正则匹配

- `^` 行首
- `$` 行尾
- `x{m}` x出现m次 ; `x{m,}` x出现m次至多次(至少m次); `x{m,n}` x出现m次至n次
- `a(bc|de)f` 表示abcf 或 adef

- `.` 除了换行符以外的任意单个字符
- `*` 前导字符的零个或多个
- `.*` 所有字符
- `[]` 字符组内的任一字符
